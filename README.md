# OSINT Labs

Welcome to my documentation and practice workspace for cybersecurity concepts. This repository tracks my hands-on learning progress.

## 🌐 Networking & OSINT

### 📘 Core Networking Concepts Learned
* **TCP/IP**: Mastered the foundational suite of communication protocols powering the internet.
* **DNS**: Explored how domain names translate into machine-readable IP addresses.
* **HTTP & HTTPS**: Analysed secure vs unsecure web traffic and encryption patterns.
* **Ports**: Studied common digital entry points and transport layers.
* **Firewalls**: Reviewed traffic-filtering mechanics used to block unauthorized network access.

### 🔍 OSINT Concepts Learned
* **Google Dorks**: Formulated advanced search operators to uncover hidden internet data.
* **WHOIS**: Extracted domain registration data, ownership timelines, and contact records.
* **DNS Lookup**: Traced network mapping records back to host infrastructures.
* **Reverse Image Search**: Used visual anchors to locate the original source of online media.
* **Metadata**: Stripped hidden data (EXIF tags, author profiles, timestamps) from files.

---

### 🧪 Practical Lab Applications
Here are the open-source tools I deployed to execute my threat intelligence research:

* [crt.sh](https://crt.sh) – Queried public Certificate Transparency logs to find subdomains.
* [DNSDumpster](https://dnsdumpster.com) – Mapped out host records and discovered attack surfaces.
* [Shodan](https://shodan.io) – Explored internet-connected IoT devices and open server ports.
* [VirusTotal](https://virustotal.com) – Analyzed suspicious URLs and files for malicious signatures.
